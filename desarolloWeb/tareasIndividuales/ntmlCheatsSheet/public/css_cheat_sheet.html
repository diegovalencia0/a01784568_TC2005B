border

El border es como ponerle un marco a un elemento. Se puede elegir el grosor, el estilo y el color del borde.

border-left

El border-left es solo el borde del lado izquierdo. Se usa cuando solo se quiere un borde en ese lado.

border-right

El border-right es el borde del lado derecho. Perfecto para darle estilo solo a ese lado.

border-top

El border-top es el borde de arriba. Ideal si se quiere un borde solo en la parte superior.

border-bottom

El border-bottom es el borde de abajo. Se usa para darle un borde solo a la parte inferior.

border-radius

El border-radius redondea los bordes. Así se pueden hacer esquinas redondeadas,

background-image

El background-image pone una imagen de fondo.

color

El color cambia el color del texto.

font

El font es una forma rápida de cambiar varias cosas de la fuente a la vez, como estilo, peso y tamaño.

font-family

El font-family cambia la familia de la fuente.

font-style

El font-style cambia el estilo de la fuente a cursiva o normal.

font-weight

El font-weight cambia el grosor del texto. Se puede hacer en negrita o más ligero.

width & height

Las propiedades width y height definen el ancho y alto del elemento. Se le puede dar el tamaño que se necesite.

list-style

El list-style cambia el estilo de las listas. Se pueden usar círculos, cuadrados, y otras figueras.

margin

El margin añade espacio fuera del borde del elemento. Se usa para separar elementos.

padding

El padding añade espacio dentro del borde del elemento para darle espacio interno.

overflow

El overflow controla cómo se muestra el contenido que se sale del tamaño del elemento.

display

El display cambia cómo se muestra un elemento.

position

El position controla cómo se posiciona un elemento en la página.

filter

El filter aplica efectos gráficos a los elementos, como hacerlos en blanco y negro, difuminarlos, etc.
